Welcome to My Website

This website provides an API that allows you to retrieve data about tourist spots in East Java. Whether you're looking for cultural sites, amusement parks, natural attractions, or religious places, our API offers a comprehensive database to help you find the best places to visit.

Our API is designed to be easy to use and integrate, offering endpoints for various categories of tourist spots. Explore the rich cultural heritage, scenic landscapes, and exciting attractions of East Java with our detailed and up-to-date information.
